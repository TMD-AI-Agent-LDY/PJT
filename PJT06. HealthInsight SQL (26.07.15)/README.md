# 헬스케어 SQLite 데이터 분석 미니 프로젝트 요구사항 명세서

## 1. 제공 데이터

이번 프로젝트에서는 아래 3개 CSV 파일을 SQLite 데이터베이스로 불러와 분석한다.

| 파일명 | 설명 | 주요 용도 |
|---|---|---|
| `patients.csv` | 환자 기본정보 | 환자 특성, BMI, 기왕력 분석 |
| `visits_2024.csv` | 2024년 진료 방문 기록 | 진료 방문 조회, 조건 검색 |
| `visits_2025.csv` | 2025년 진료 방문 기록 | 진료 방문 조회, JOIN 분석 |

데이터 규모:

- `patients.csv`: 1,200행, 14열
- `visits_2024.csv`: 1,673행, 13열
- `visits_2025.csv`: 1,927행, 13열

---

## 2. 사전 준비

본격적인 작업에 앞서 다음 작업을 수행하시오.

1. 프로젝트 폴더에서 SQLite 데이터베이스 파일 `healthcare.sqlite3`를 생성하시오.
2. CSV Load 기능을 사용하여 아래 CSV 파일을 테이블로 불러오시오.

| CSV 파일 | 테이블명 |
|---|---|
| `patients.csv` | `patients` |
| `visits_2024.csv` | `visits_2024` |
| `visits_2025.csv` | `visits_2025` |

테이블명은 반드시 위 표와 동일하게 사용한다.

---

## 4. 제출물

다음 내용을 포함하여 제출하시오.

1. SQL 쿼리 파일 1개
2. 각 요구사항별 실행 결과 캡처 또는 결과 표

## 5. 필수 구현 요구사항

### 요구사항 1. 테이블 Load 결과와 기본 조회를 확인하시오.

3개 테이블이 정상적으로 생성되었는지 확인하기 위해 각 테이블에서 앞 5개 행을 조회하시오.

필수 쿼리:

- `patients` 앞 5개 행 조회
- `visits_2024` 앞 5개 행 조회
- `visits_2025` 앞 5개 행 조회

---

### 요구사항 2. SELECT, ORDER BY, LIMIT 를 활용해 데이터 조회를 구현하시오.

1. `patients` 테이블에서 `환자ID`, `성별`, `나이`, `지역`, `BMI` 컬럼만 조회하시오.
2. `patients` 테이블에서 `BMI`가 높은 환자 상위 10명을 조회하시오.
3. `visits_2025` 테이블에서 `진료비_원`이 높은 방문 기록 상위 10건을 조회하시오.

---

### 요구사항 3. WHERE, IN, LIKE를 사용해 필터링을 구현하시오.

1. `patients` 테이블에서 `BMI`가 25 이상인 환자를 조회하시오.
2. `patients` 테이블에서 `지역`이 서울 또는 경기인 환자를 조회하시오.
3. `visits_2025` 테이블에서 `진료과`가 내과 또는 응급의학과인 방문 기록을 조회하시오.
4. `visits_2025` 테이블에서 `주증상`에 `기침`이라는 단어가 포함된 방문 기록을 조회하시오.

---

### 요구사항 4. JOIN으로 환자 정보와 방문 정보를 연결하시오.

`visits_2025`와 `patients`를 `환자ID` 기준으로 JOIN하여 환자 정보가 포함된 방문 기록을 조회하시오.

다음 컬럼을 출력하시오.

- `방문ID`
- `방문일`
- `진료과`
- `방문유형`
- `진료비_원`
- `성별`
- `나이`
- `지역`
- `BMI`

추가로 다음 조회를 수행하시오.

1. JOIN 결과에서 `BMI`가 25 이상인 환자의 방문 기록을 조회하시오.
2. JOIN 결과에서 `진료비_원`이 높은 방문 기록 상위 20건을 조회하시오.

---

### 요구사항 5. GROUP BY와 집계 함수를 사용하시오.

1. `patients` 테이블에서 지역별 환자 수를 구하시오.
2. `visits_2025` 테이블에서 진료과별 방문 건수를 구하시오.
3. `visits_2025` 테이블에서 진료과별 총 진료비를 구하시오.
4. `visits_2025` 테이블에서 방문유형별 평균 진료비를 구하시오.

힌트:

- `COUNT(*)`
- `SUM(컬럼명)`
- `AVG(컬럼명)`
- `GROUP BY`
- `ORDER BY`

2. `UNION ALL`로 합친 전체 방문 데이터에서 진료과별 방문 건수를 구하시오.
